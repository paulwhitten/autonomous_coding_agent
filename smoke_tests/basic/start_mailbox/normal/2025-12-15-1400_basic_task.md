# Create sum module with tests

Create a TypeScript module that exports a `sum` function, write Jest tests for it, and verify the tests pass.

## Requirements

1. **Create `sum.ts`** in the project root directory:
   - Export a function `sum(a: number, b: number): number`
   - Returns the sum of `a` and `b`

2. **Update `package.json`** so the `test` script runs Jest:
   - Set `"test": "jest"` in the scripts section

3. **Create `sum.test.ts`** in the project root directory:
   - Import `sum` from `./sum`
   - Test cases:
     - Two positive numbers (e.g., `sum(2, 3)` returns `5`)
     - Mixed positive and negative (e.g., `sum(-2, 3)` returns `1`)
     - Two negative numbers (e.g., `sum(-2, -3)` returns `-5`)
     - Zero as an argument (e.g., `sum(0, 5)` returns `5`)

4. **Run `npx jest` from the project root** and confirm all tests pass

## Acceptance Criteria

- `sum.ts` exists and exports a typed `sum` function
- `sum.test.ts` exists with at least 4 test cases
- `npx jest` exits with code 0 and reports all tests passing
- `package.json` scripts.test is set to `"jest"`
