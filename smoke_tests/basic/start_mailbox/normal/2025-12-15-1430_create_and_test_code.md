# Basic Task - Create and Test Simple Code

Create a simple TypeScript utility function, write tests, and verify it works.

## Task Requirements

1. Create a file `workspace/utils.ts` with a simple utility function:
   - Function name: `addNumbers`
   - Takes two numbers as parameters
   - Returns their sum
   - Include type annotations

2. Create a test file `workspace/utils.test.ts`:
   - Import the function
   - Write 3 test cases:
     * Positive numbers
     * Negative numbers
     * Zero

3. Run the tests and verify they pass

4. Create a summary file `workspace/test_results.txt` with:
   - Test results
   - Function created
   - All tests passing

## Success Criteria

- `workspace/utils.ts` exists with typed function
- `workspace/utils.test.ts` exists with 3 tests
- `workspace/test_results.txt` exists
- All tests pass
